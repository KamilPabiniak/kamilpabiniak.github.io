    • Material&Mesh Replacer - editor script

      Chesspeare
      Show info

      This script was used by me and my team to quickly change materials and possibly prefab mesh on selected scenes, so that we don't have to switch to each scene. Although a similar effect can be achieved with the prefab itself, once we have the same prefab on each scene we still have to change it everywhere.

      This tool was just meant to make the process easier, so that graphic designers can quickly examine whether a particular texture or mesh is appropriate and do all the work on all the scenes with literally 4/5 clicks.

      I am working on a script to add a function to automatically add a new prefab of changed objects and automatically set it on each scene.
      Work in progress
      The instructions for this tool are as follows:

      1. Prefab to Search:
      - Select the prefab you want to search.

      2. Material to Replace With:
      - Select the material from the project resources that you want to assign to the prefab.

      3. Replace Mesh (optional):
      - Check the 'Replace Mesh' option and select the mesh from the project resources that you want to assign to the prefab.

      4. Scenes to Search:
      - Select the scenes you want to search.

      5. 'Replace Material and Mesh' Button:
      - After selecting the prefab, material, mesh (optional), and scenes, click the 'Replace Material and Mesh' button to start the process of replacing the material and/or mesh.
      - Replacing material and/or mesh in prefab instances:
      - The tool searches the selected scenes and replaces the material and/or mesh in all instances of the prefab.

      Example:
      Let's say you want to replace the material CatMaterial and the mesh CatMesh in all instances of the CatPrefab in the selected scenes:
      1. Select CatPrefab in the 'Prefab to Search' field.
      2. Select CatMaterial in the 'Material to Replace With' field.
      3. Check 'Replace Mesh' and select CatMesh in the 'Mesh to Replace With' field.
      4. Select the appropriate scenes in the 'Scenes to Search' section.
      5. Click the 'Replace Material and Mesh' button.

    • Grid Position Set - script

      Chesspeare
      Show info

      This is a script I created myself to easily set levels in our Chesspeare game. The most important element is **On Grid Position.** This is a tool that allows you to conveniently create figure settings for given levels. Initially the fields look like this:

      The yellow box

      is our chosen field in the editor. Black is the initial position of our King. The white boxes are those already occupied by enemy figures and the blue boxes are special boxes. An empty field has an unchecked "*Set Chess*" box. We can then choose whether to place a figure or a special field on that field.

      In the blue area

      we must have ticked: **Chess/Objects**: Chess (i.e. an indication that we are placing a figure). **Choose colour**: White (if we are placing an enemy figure) or Black (if our King). **Choose chesstype**: Here we indicate the specific type of figure we want.

      The green area

      is **Moves Data**. For all figures except pawns (*Pawn*) we need to fill in this area. It contains a list of the subsequent moves that the figure will want to make. Subsequent moves are described by changes on the X/Z axis in relation to the figure's current position on the board. Note: this is the move the figure wants to make. It may differ from the actual one if the figure's movement is affected by elements such as special fields, knockdown or special rules of the world.

      The setting up of special fields is still included in the script and I would be happy to talk about how this section works in person.
